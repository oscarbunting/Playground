Creating Variables

We can use either let or const to define variables. Variable names cannot container spaces, but they can contain numbers. when we need to combbine more than one word, we use camel Case, leaving the 1st word lowercase and capitalising the 1st letter of each additional word.
For example: let word1Word2

Storing values with variables

After creating a variable, we can use it to store data, in this case we call that data a Value.
We use an equal sign to show which data to store within, for example:
const userCity = "Reigate";
In the example above, the type of data we're storing is called a string, a string can contain numbers and letters with spaces and special charecters.

Differences between let and const

Const, which stands for constant, is used to declare variables that aren't supposed to change, for example:
const englandCapital = "London";

To make it extra clear that a variable is constant and shouldn't chnage, we can name it in all caps, for example:
const BIRTHDAY = "23/08/03"

We can't use camelCase if we use the uppercase standard, so we use snake_case instead when we make the variable name all caps. If we try and update a constant variable, we will throw an error. On the other hand, variables that are declared using let can be updated by using the assign = operator. For example we could create a variable like this:
let oscarAge = 20
And then update the variable at a later time like this:
oscarAge = 21

We'll need to use both let and const, and in theory we could use them interchangeably. However, we should try to use const by default

Building Instructions

When we write lines of code, we're giving the computer instructions, and the order in which we give the computer those instructions is important. In order to see what our instructions did, we can output the result to the console using the console.log instuction. For example: console.log("Hello, World"); would output Hello, World. We can us console.log as often as we'd like. The computer will display the output on a line in the console. We can also use console.log to out put the value inside a variable. For example:
const oscar = "Bunting";
console.log (oscar);

Output/Console:

"Bunting"
