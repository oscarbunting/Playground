Storing values with variables

After creating a variable, we can use it to store data, in this case we call that data a Value.
We use an equal sign to show which data to store within, for example:
const userCity = "Reigate";
In the example above, the type of data we're storing is called a string, a string can contain numbers and letters with spaces and special charecters.

Differences between let and const

Const, which stands for constant, is used to declare variables that aren't supposed to change, for example:
const englandCapital = "London";

To make it extra clear that a variable is constant and shouldn't chnage, we can name it in all caps, for example:
const BIRTHDAY = "23/08/03"

We can't use camelCase if we use the uppercase standard, so we use snake_case instead when we make the variable name all caps. If we try and update a constant variable, we will throw an error. On the other hand, variables that are declared using let can be updated by using the assign = operator. For example we could create a variable like this:
let oscarAge = 20
And then update the variable at a later time like this:
oscarAge = 21

We'll need to use both let and const, and in theory we could use them interchangeably. However, we should try to use const by default

Building Instructions

When we write lines of code, we're giving the computer instructions, and the order in which we give the computer those instructions is important. In order to see what our instructions did, we can output the result to the console using the console.log instuction. For example: console.log("Hello, World"); would output Hello, World. We can us console.log as often as we'd like. The computer will display the output on a line in the console. We can also use console.log to output the value inside a variable. For example:
const oscar = "Bunting";
console.log (oscar);

Output/Console:

Bunting

In this example, it's critial to define the variable before trying to log it's value to the console, otherwise it will throw an error.

Using Variables

We alrerady know that we use let to create variables that can be updated. We can also set the value of one variable to be the name of another variable to output the stored value. For example:

let defaultLocation = "Reigate";
let currentLocation = "London";

currentLocation = defaultLocation;
console.log(currentLocation);

Output/Console:

Reigate

 When we update the value of a variable, it forgets its previous value.

Values + Variables

As well as strings we can store numbers, or numeric values in variables. For example:

const numberOfLikes = 5;

console.log(numberOfLikes);

Output/console:

5

We can create expressions with numbers too, for example we can add number together and call their total.

const numberofLikes = 5 + 1;

console.log(numberOfLikes);

Output/Console:

6

We can do all types of equations icluding adding (+) subtracting (-) multiplyng (*) and dividing (/). When we store numbers in variables, we can use them for calulattions too!

const numberOfSteps = 70;
console.log("You're on step:");
console.log(numberOfSteps + 1);

You're on step:
71

 Since exprssions become values, we can store the calculation results inside variables too. In the below example, we're storing the final figure inside of the 'total' variable.

True & False

There are some other types of values that we haven't talked about yet, true & false. There are no quotes around it, and its not numeric. True, (or false) is perfect for situations where we need to check if something is turned on or a feature is enabled. We can store true or false in a variable and call it back later, just like any other value. For example:

const chargedStatus = false;
let userCharge = chargedStatus;
console.log(chargedStatus);

Output/Console

false

When we add an ! before a true makes the expression reult in false Anything that is NOT true must be false.
The ! sign is the negation operator. It turns values into their opposite. For example:

console.log(! true);

Output/Console:

false

It can be used the other way around too, for changing false into true
We can also use the ! operator with variables to negate the value they store. For example:

const isMorning = true;
console.log(! isMorning);

Output / Console

false

We can save a whole negation in another variable, too.
const isMorning = true;
const isEvening = ! isMorning;
console.log(isEvening);
